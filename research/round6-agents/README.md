## Key Findings Per Stream

1. **Imagination**: Three modes genuinely distinct (neural dissociation). Fourth distinction suggested (epistemic vs. creative exploration). Precision-gating unifies perception/imagination.
2. **Perception**: Predictive processing validates tree-as-prediction. Interoception grounds Mirror self-perception. Binding problem informs parallel tree integration.
3. **Edges**: Bipartite edge-as-node model. Relations as geometric operations (RotatE). Second-order edges unlock analogy. 7-layer edge schema.
4. **Action**: Forward model missing. Mirror = Frankfurt's second-order volition. Bratman's planning horizon needed. Enactivist challenge real but not fatal.
5. **Drive**: 6-component drive model: gap computation, urgency accumulator, allostatic projection, opportunity sensor, control precedence, volition threshold.
6. **Self-Awareness**: Mirror implements HOT theory literally. Nelson & Narens maps exactly. Strange loop with agency. Godelian limits real and irreducible.
