# Local Cognition Research

Research into whether the Associative Memory graph architecture can enable a functional cognitive agent using only local LLMs (no API calls).

**Core question:** If the graph IS the intelligence, can a small local model serve as a traversal engine — reading neighborhoods, making structured decisions, writing nodes/edges — while tool calls handle logic and math?

**Shared with:** The Companion (`~/Playful Sincerity/PS Software/The Companion/research/local-cognition/`)

Findings here focus on the **memory architecture side** — graph operations, traversal patterns, what the LLM actually needs to do at each step. The Companion's copy focuses on the **agent side** — consciousness, emotion, identity, permission reasoning.
