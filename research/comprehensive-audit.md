## 1. The Claim

The Associative Memory Architecture proposes a novel cognitive substrate for AI agents where:

- The memory graph IS the interpretable world model, replacing the role pretrained weights play in forming beliefs
- Active cognition is trees growing through the graph (not retrieval-then-load)
- A persistent conscious evaluation loop (the Mirror) watches, steers via emotions, and learns
- Associations form through experiential co-occurrence (Hebbian/PMI), not pretrained embedding similarity
- The LLM is a reasoning engine; the graph is the mind
- Different agents build different experiential topologies = different worldviews, readable and diffable

The architecture operates on three planes: the Matrix (long-term memory, one graph), Trees (short-term working memory, ephemeral growing subgraphs), and the Mirror (persistent consciousness that observes and steers the subconscious).

## 3. Round-by-Round Findings

### Round 1 — Landscape Survey (6 agents)
**Question:** Does anything like this exist?

**Scope:** 50+ systems across 6 dimensions: cognitive science foundations, existing agent memory systems, graph-based retrieval, semantic embeddings, dynamic context management, neural memory architectures.

**Key findings:**
- No production system combines associative traversal + dynamic graph context + experiential distance + anti-hallucination by design
- Closest systems: SYNAPSE (spreading activation, paper only), HippoRAG 2 (PPR over KG), A-MEM (Zettelkasten linking)
- Maximum properties any modern system implements: ~4 out of 10
- Strong theoretical foundations exist: ACT-R, spreading activation, GWT, CLS, hippocampal indexing
- Key mechanism validated: PMI = Hebbian learning (BCPNN proof). PPR = spreading activation (IEEE 2021 proof).

**Conclusion:** The gap is real. The individual mechanisms are established. The integration is novel.

---

### Round 2 — Targeted Mechanism Research (5 agents)
**Question:** How do the specific open design questions resolve?

**Scope:** Experiential distance metrics, anti-hallucination architectures, consciousness pointer mechanisms, embedding drift + graph evolution, interpretable world models.

**Key findings:**
- **Experiential distance:** Edge weight formula: PMI × salience × temporal proximity × decay. BCPNN benchmarks 3x better than alternatives.
- **Anti-hallucination:** Best grounding achieves ~86% compliance (FACTS benchmark). GCR/KG-Trie achieves 100% for structured KG paths. Free-form synthesis remains irreducible leakage point. Smaller models = less leakage.
- **Consciousness pointer → retired.** Replaced by "trees growing within the matrix." PPR seed node = tree root. Focus-biased entry points. SOAR-style stack for subgoaling.
- **Embedding drift:** TGN-style message passing updates embeddings when new edges form. Memory reconsolidation "largely uninvestigated" in AI (2025 survey).
- **Interpretable world models:** WorldCoder (NeurIPS 2024) proves readable world models work. AriGraph (IJCAI 2025) closest to "KG as world model." Pearl's causal hierarchy requires three edge types.

**Conclusion:** All 5 open questions answered with concrete mechanisms. The "consciousness pointer" concept retired in favor of the three-plane model.

---

### Round 3 — Full System Integration Audit (5 agents)
**Question:** Does the complete integrated system (Matrix/Trees/Mirror + value system) exist anywhere?

**Scope:** Tree-as-context rendering, meta-cognitive same-substrate, parallel cognitive streams, value-modulated cognition, property matrix across 25+ systems.

**Key findings:**
- **Tree-as-context:** Zero implementations. Sits in gap between MemTree (grows right, flattens) and ReMindRAG (subgraph IS context, but linear). VERSES Active Inference is the deepest theoretical match.
- **Same-substrate metacognition:** NARS closest (same substrate for self/world). CLARION has meta-cognitive subsystem. 2025 CMC proposal explicitly calls for this. Nobody implements meta-layer spawning exploration from self-observation.
- **Parallel trees:** No system satisfies all criteria (single agent + shared memory + parallel growing trees + shared budget). LIDA/GWT is closest but enforces serialization bottleneck.
- **Value-modulated cognition:** MAMID is closest for emotion-as-architecture. Dual-axis confidence (comprehension × completeness) appears nowhere. Care as experientially grounded (not formula-computed) is novel.
- **Integration matrix:** Maximum any system achieves: ~5.5/10 (ACT-R, pre-LLM). Best modern: ~4/10 (Hindsight). Properties C, D, G, I have zero full implementations anywhere.

**Conclusion:** 7 specific features have zero implementations. The three-plane model has no precedent. The combination of even 5 properties has never been attempted.

## 8. Strongest Counterarguments (Steel-Manned)

> Detailed arguments in: [stream-6](round4-agents/stream-6-agi-assessment.md) (all 5 objections), [stream-2a](round4-agents/stream-2a-consciousness-philosophy.md) (consciousness limits), [stream-2b](round4-agents/stream-2b-agi-definitions.md) (AGI criteria)

### 1. Connectionist Objection
**"You're trading the most powerful representational innovation in AI for interpretability."** Distributed representations capture graded similarity, context-sensitivity, and novel combinations that explicit graphs cannot.
**Our response:** The architecture delegates distributed reasoning to the LLM. The graph provides structured knowledge; the LLM composes it. The tradeoff is real but conscious — transparency is worth the expressiveness cost for the use cases this targets (auditable, trustworthy agents).

### 2. Embodied Cognition Objection
**"Without a body, you have sophisticated symbol manipulation, not genuine understanding."**
**Our response:** Recent research argues "intelligence requires grounding but not embodiment." The architecture grounds through tool use, API interaction, and multimodal nodes. Weaker than physical grounding, but sufficient for digital environments. The multimodal extension (images, sounds, sensors) is the pathway to richer grounding.

### 3. Scaling Objection
**"o3 scored 87.5% on ARC-AGI-1 through scale alone. Why build elaborate architecture?"**
**Our response:** Scaling produces capability but not interpretability, personalization, or value alignment. Different goals. Also: ARC-AGI-2 dropped scores to 24-54%, and scaling hits diminishing returns. The architecture competes on transparency, not task performance.

### 4. Cognitive Science Objection
**"You're implementing metaphors, not mechanisms. 'Consciousness' and 'emotions' are heavyweight terms for engineering constructs."**
**Our response:** Acknowledged. We use functional-analogy framing explicitly. The Mirror implements access consciousness (GWT), not phenomenal consciousness. Emotions are "value-alignment signals that modulate processing," not felt experiences. This is standard practice in cognitive architecture research (ACT-R, Soar, CLARION all do this).

### 5. Historical Precedent Objection
**"Every decade has had 'this is the path to AGI.' Expert systems, neural nets, transformers. What makes this different?"**
**Our response:** We explicitly do NOT claim AGI. The architecture targets transparency and value alignment, not general capability. The claim is: "a new substrate for building AI agents whose beliefs can be read, compared, and corrected." That is a concrete, testable contribution — not a general intelligence claim.

**Question:** How does the agent perceive, act, learn skills, and build causal knowledge from world interaction?

**Raw agent outputs:** All detailed findings saved in [round5-agents/](round5-agents/). See [round5-agents/README.md](round5-agents/README.md) for the full index.

**Scope:**

| Stream | Agents | What Was Investigated | Raw Output |
|--------|--------|----------------------|------------|
| 1: Procedural Memory | 1 | ACT-R, SOAR, CLARION, NARS, Mnemos | [stream-1](round5-agents/stream-1-procedural-memory.md) |
| 2: Action-Perception | 1 | Active inference, VERSES RGM, Gibson affordances, enactivism, predictive coding | [stream-2](round5-agents/stream-2-action-perception-loop.md) |
| 3: Tool Use Patterns | 1 | ReAct, Toolformer, LATS, Voyager, Reflexion, CREATOR, ToolBench, AriGraph | [stream-3](round5-agents/stream-3-tool-use-patterns.md) |
| 4: Skill Learning | 1 | Dyna, Options framework, MAML, Voyager, SkiMo, GraphICL | [stream-4](round5-agents/stream-4-skill-learning.md) |
| 5: Causal Discovery | 1 | Pearl hierarchy, GFCI, PCMCI+, NOTEARS, ACE, CauseNet, CausalKG | [stream-5](round5-agents/stream-5-causal-discovery.md) |

**Key findings:**

1. **Action IS tree growth** — confirmed by active inference (Friston). Not a metaphor — formally the same process as perception.
2. **VERSES RGM** — "only active pathways require belief updating" = our spreading activation. Deepest theoretical match with specific paper (arXiv:2407.20292).
3. **Skill acquisition = consolidation** — ACT-R compilation, SOAR chunking, and Dyna replay all converge: repeated (context → action → outcome) patterns compile into procedural knowledge during consolidation.
4. **NARS validates unified graph** — no procedural/declarative distinction. Skills, knowledge, and self-model in one network.
5. **Universal tool-use gap confirmed** — 13 modern AI agent systems surveyed; none integrates tool experience into an associative graph. Voyager closest but flat vector store.
6. **Causal discovery is feasible** — GFCI + PCMCI+ pipeline runs in under 1 minute during consolidation. ~20-30 observations per (tool, context) pair. Tool use IS the do-operator.
7. **5 cross-stream convergences** — action IS cognition, prediction error is universal, consolidation = compilation = discovery, Mirror is formally grounded, unified graph is validated.

**Design outputs:** SkillNode data structure, affordance edges, causal edge pipeline, Goal nodes, SELF cluster, skill selection equation, two-level activation, Mirror dual evaluation (epistemic + pragmatic). All 6 open design questions resolved or partially resolved.

**Full synthesis:** [round5-synthesis.md](round5-synthesis.md)

### Round 6 — Conceptual Foundations (6 agents, 6 streams)
**Question:** What are the philosophical and cognitive-scientific foundations of the architecture?

**Scope:** Imagination/simulation, perception as construction, edge semantics/relations, philosophy of action, drive/motivation, self-awareness/recursion. 150+ papers, 10+ primary text books read.

**Key findings:**

| Stream | Core Discovery | Architecture Impact |
|--------|---------------|-------------------|
| 1 - Imagination | Three modes genuinely distinct; precision-gating unifies perception/imagination | Add `sensory_precision` parameter (0.0-1.0) to trees |
| 2 - Perception | Perception IS prediction + error; Mirror is interoceptive predictor | Tree processes `input XOR prediction`; affect from error integration |
| 3 - Edges | Relations are first-class geometric operations; second-order enables analogy | Bipartite edge-as-node model; 7-layer edge schema; RotatE operations |
| 4 - Action | Forward model missing; Mirror = second-order volition; planning horizon needed | predict→act→compare→update loop; Bratman planning layer |
| 5 - Drive | 6-component drive model: gap + urgency + allostasis + opportunity + precedence + Rubicon | Explicit gap computation; volition threshold; urgency accumulator |
| 6 - Self-Awareness | Mirror implements HOT literally; strange loop with agency; Godelian limits real | Nelson & Narens maps exactly; meta-memories need confidence weights |

**Seven cross-stream convergences:** simulation all the way down, forward model as requirement, first-class edges enable everything, Mirror validated from 8 frameworks, thinking-to-acting gap is a design problem, three modes + precision-gating, same-substrate validated by all frameworks.

**Design output:** 6 must-add mechanisms, 5 must-clarify decisions, 6 confirmed-as-is. See round6-synthesis.md.
